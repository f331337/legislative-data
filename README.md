# Legislative Data Processing

This project processes publicly available legislative data and generates summary
reports for legislators and bills based on voting results.

## Requirements

- Python **3.9.6+**
- No external dependencies

## Running the Application

From the project root, run:

```bash
python3 main.py
```

## Tests
```bash
python3 -m unittest discover -s src/tests -t src -p "test_*.py"
```

## Output
Generated files saved in output/ folder
